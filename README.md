# LOL-Voice-Chat (롤보챗)
<br/>

## 💁‍♂️ 서비스 소개
[리그오브레전드](https://www.leagueoflegends.com/ko-kr/) 유저들의 바쁜손은 플레이에만 집중!  
그냥 켜놓으면 롤에서 매칭된 팀원과 음성채팅이 가능하게되는 데스크탑 어플리케이션 서비스

**롤 유저들의 불편함** : 랜덤으로 5명의 팀원이 매칭되는데 팀원간 음성채팅은 지원하지않음.  → 롤보챗으로 해결 ✨

지원하는 운영체제 : window, mac os
